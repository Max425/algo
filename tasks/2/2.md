Что выведет программа?

```go
package main

import (
	"fmt"
)

func a() {
	x := []int{}
	x = append(x, 0)
	x = append(x, 1)
	x = append(x, 2)
	y := append(x, 3)
	z := append(x, 4)
	fmt.Println(y, z)
}

func main() {
	a()
}
```

Решение:

1. Рассказать про устройство слайсов
2. `fmt.Println(y, z)` - выведет [0 1 2 4] [0 1 2 4]
3. Объяснить, что это связанно с тем, что базовый массив у них один
